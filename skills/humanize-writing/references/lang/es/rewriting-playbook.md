# Rewriting playbook — Spanish (Español)

Concrete fix recipes per category. Read alongside `es/ai-tell-taxonomy.md`.

## General order
1. Remove S1s first. 2. Thin S2 clusters. 3. Leave lone S3s.
4. Read aloud. If it sounds like a government report, you've left too many ES-A words.

---

## ES-A léxico (vocabulary)
- crucial/fundamental → importante / clave / decisivo depending on context
- implementar → aplicar / poner en práctica / usar
- optimizar → mejorar / afinar — only keep optimizar in technical contexts where it's exact
- robusto → sólido / fiable
- potenciar/empoderar → fortalecer / impulsar / dar más control a
- Delete the adjective if the noun stands alone: "solución robusta" → "solución"

## ES-B aperturas (openers)
- Delete "En el mundo actual / En el panorama actual" entirely — start on the first real idea
- "Es importante destacar que" → delete the phrase; the claim that follows stands on its own
- "Cabe mencionar que" → delete

## ES-C antítesis (antithesis)
- "No se trata solo de X, sino de Y" → state the primary claim: "Es una herramienta que [efecto concreto]"
- Keep only if the contrast is genuinely the point (rare)

## ES-D conectores (connectives)
- En primer lugar/En segundo lugar stacked → prose flow: "Primero… Luego… Por último…" used sparingly, or restructure without enumeration
- Asimismo/Por otro lado reflexively → delete; let the logic connect implicitly
- "Es importante mencionar que" as filler intro → delete

## ES-E cierres (conclusions)
- En conclusión/En resumen → delete; end on the last real point
- "Esto demuestra la importancia de X" → delete; the argument already demonstrated it

---

## Change-rate: >30% warn, >50% stop.
## Failure modes: trading academic words for other academic words; false informality (tuteo where usted fits).
