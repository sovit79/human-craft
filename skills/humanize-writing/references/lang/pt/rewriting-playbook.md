# Rewriting playbook — Portuguese / Brazilian (Português Brasileiro)

Concrete fix recipes per category. Read alongside `pt/ai-tell-taxonomy.md`.

## General order
1. Remove S1s first. 2. Thin S2 clusters. 3. Leave lone S3s.
4. Read aloud in BR Portuguese. If it sounds like a formal report for a government agency, register is off.

---

## PT-A vocabulário (vocabulary)
- crucial/fundamental → importante / decisivo / essencial depending on context
- alavancar → usar / aproveitar / impulsionar
- nortear → guiar / orientar / definir (nortear is literary; often simpler word fits)
- fomentar → incentivar / estimular / apoiar
- capacitar → treinar / preparar / habilitar
- robusto → sólido / confiável / estável
- Delete the adjective if the noun stands alone: "solução robusta" → "solução"

## PT-B aberturas (openers)
- "No cenário atual / No mundo contemporâneo" → delete; start on the first real idea
- "Vale ressaltar que / Cabe destacar que" → delete; the claim that follows stands alone
- "É importante mencionar que" → delete

## PT-C antítese (antithesis)
- "Não se trata apenas de X, mas de Y" → state the primary claim: "Esta ferramenta [efeito concreto]"

## PT-D conectivos (connectives)
- Em primeiro lugar / Em segundo lugar stacked → prose flow or restructure without enumeration
- Além disso / No entanto reflexively → delete or use "e / mas / então"
- "É válido ressaltar que" as filler intro → delete

## PT-E mistura EU/BR (register mixing)
- Audit for EU-PT words: utilizador → usuário, telemóvel → celular, ecrã → tela, autocarro → ônibus
- EU spelling: óptimo → ótimo, acção → ação, facto → fato
- Pick one register and apply consistently throughout

## PT-F encerramentos (conclusions)
- Em conclusão / Em suma / Sendo assim → delete; end on the last real point
- "Isso demonstra a importância de X" → delete; the argument already demonstrated it

---

## Change-rate: >30% warn, >50% stop.
## Failure modes: mixing EU vocabulary after fixing some BR terms; replacing formal words with other formal words.
