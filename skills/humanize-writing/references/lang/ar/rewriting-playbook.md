# Rewriting playbook — Arabic (العربية)

Concrete fix recipes per category. Read alongside `ar/ai-tell-taxonomy.md`.

## General order
1. Remove S1s first. 2. Thin S2 clusters. 3. Leave lone S3s.
4. Read aloud. If every sentence uses a masdar (verbal noun) where a verb would do, you've over-formalised.

---

## AR-A سجل وأسلوب (register/vocabulary)
- فضلاً عن ذلك / ومن الجدير بالذكر → delete; start on the actual point
- وتجدر الإشارة إلى أن → delete; if the information is worth saying, say it directly
- يكتسب أهمية بالغة → say why it matters specifically: "يحدد هذا القرار مسار المشروع"
- لا شك أن → delete; state the claim without the preamble
- Excessive masdar (verbal nouns) where verbs are more direct: "إجراء تحليل" → "تحليل" or "نحلّل"

## AR-B افتتاحيات (openers)
- "في عصرنا الحالي / في ظل التطور المتسارع" → delete; start on the first real idea
- "يسعدنا أن نستعرض" → delete; begin the content directly

## AR-C ليس فقط بل أيضاً (antithesis)
- Pick the core claim: "هذه الأداة تُعيد تشكيل طريقة عملنا" not "ليس هذا مجرد أداة، بل هو منهج متكامل"
- The Arabic antithesis construction is especially strong — one use signals AI immediately

## AR-D روابط الجمل (connectives)
- أولاً/ثانياً/ثالثاً stacked → prose paragraphs; use the order itself to imply sequence
- وأيضاً/وكذلك/وعلاوة على ذلك repeating → delete; Arabic flows naturally without dense connectives
- من ناحية أخرى reflexively → keep only for a genuine reversal

## AR-E خواتيم (conclusions)
- خلاصة القول / وفي الختام → delete; end on the last real point
- "وهذا ما يجعل X ضرورةً لا غنى عنها" → delete; the argument already made the case

---

## Change-rate: >30% warn, >50% stop.
## Failure modes: replacing formal MSA with dialect words inappropriately; over-simplifying to the point of ambiguity.
