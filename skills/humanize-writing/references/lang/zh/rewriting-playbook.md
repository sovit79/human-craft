# Rewriting playbook — Chinese Simplified (中文简体)

Concrete fix recipes per category. Read alongside `zh/ai-tell-taxonomy.md`.

## General order
1. Remove S1s first. 2. Thin S2 clusters. 3. Leave lone S3s.
4. Read aloud. If every sentence has the same cadence, you've introduced ZH-D or ZH-H.

---

## ZH-A 词汇 (vocabulary)
- 至关重要/尤为重要 → say why it matters: "这一步决定了项目成败" not "这一步至关重要"
- 赋能/助力 → describe the actual mechanism: "让团队自主决策" not "赋能团队"
- 彰显 → 说明/展示/体现 depending on what it actually does
- 深度 as empty modifier → delete or replace with a specific claim: "深度分析" → "逐行分析了每个错误"
- 诸多/各方面/全方位 → count or name them; delete if nothing is specified

## ZH-B 套话开头 (openers)
- Delete the entire opening sentence. Start on the first real idea.
- If "在当今社会" was providing context, fold that into the first sentence: "企业数字化转型已成刚需——" not "在当今数字化时代，企业数字化转型……"

## ZH-C 不只是X更是Y (antithesis)
- Pick the true claim: "这工具改变了我们设计流程" not "这不仅是一个工具，更是一种设计哲学"
- "不是…而是…" once is acceptable; as the default sentence shape → restructure

## ZH-D 连接词 (connectives)
- 首先/其次/最后 stacked → convert to prose; use paragraph breaks for sequence
- 此外/另外/除此之外 → delete; the next sentence stands alone
- 然而/但是 reflexively alternating → keep the one that marks a genuine turn; cut the rest

## ZH-E 结论套话 (conclusions)
- 综上所述/总而言之 → delete; end on the last real point
- "这正是X的价值所在" → delete the whole sentence; the argument already made the case

---

## Change-rate: >30% warn, >50% stop.
## Failure modes: swapping formal words for other formal words; MSA-style register creep.
